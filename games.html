Game Descriptions

Minesweeper: A simple implementation of the classic minesweeper game.

Black Hole Tycoon: An unfinished idle game with a periodic table theme where you click the red button to gain mass which you can use to buy elements and upgrades to cause you to gain mass faster until you create a black hole.

Bouncing Balls: A simple implementation a falling array of blocks and bouncing balls.

Bounce 3: An arkanoid-style survival game where you try to break as many blocks as you can before they reach the bottom line.

Bounce 4: The same as Bounce 3, but with more features.
