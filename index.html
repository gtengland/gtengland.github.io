Project Descriptions

Thin Film Sparklines Displays an array of graphs of thin-film reflection spectra for visualization of trends using angular.js.

Opal Sparklines Displays an array of opal/inverse opal reflection spectra for visualization of trends using angular.js.

Bragg Stack Color Dynamic calculation of the visible reflectance spectrum of the normal reflectance from an arbitrary dielectric stack.

Opal/Inverse Opal Color Dynamic calculation of the visible reflectance spectrum of the normal reflectance from a <111> opal.

Thin film interference: Dynamic calculation of the visible reflectance spectrum of a 2-layer thin film on a thick substrate.

Some simple games made in d3.
